# Инструкция менеджера

## Вход

1. Откройте приложение в браузере.
2. Войдите через Google или через локальный dev-вход, если он включен на тестовом стенде.
3. Если email не добавлен администратором в список разрешенных, вход будет запрещен.

## Создание КП

При открытии формы справа сразу отображается быстрый предпросмотр. Он обновляется во время заполнения полей и не создаёт DOCX или PDF. Кнопка `Сформировать точный PDF` нужна только для проверки окончательного фирменного оформления.

1. Нажмите `Создать КП`.
2. Выберите шаблон `КП ООО «Бештау Электроникс»`.
3. Заполните адресата:
   - полное наименование;
   - ИНН;
   - адрес, если нужен;
   - регистр отображения.
4. Укажите дату КП и ручную часть исходящего номера.
5. Выберите основание:
   - `КП без номера запроса`;
   - `КП по запросу`.
6. Для КП по запросу заполните номер и дату запроса. Поле `Основной текст` автоматически покажет текст, который попадет в документ.
7. Добавьте товары через `Добавить строку`.
8. Укажите количество и цену за единицу с НДС.
9. Проверьте автоматически рассчитанные итог и НДС.
10. Заполните сроки и дополнительные условия.
11. При необходимости очистите поле `Текст приложения/спецификации`; тогда этот абзац полностью исчезнет из итогового документа.

## Предпросмотр и генерация

1. Нажмите `Сохранить`.
2. Нажмите `Обновить предпросмотр`.
3. Проверьте PDF в правой части страницы.
4. Нажмите `Сгенерировать DOCX и PDF`.
5. Скачайте нужный файл отдельной кнопкой.

## История, дублирование и удаление

- Менеджер видит только свои КП.
- Кнопка `Дублировать` создает новое КП с теми же данными, но без старых финальных файлов.
- Кнопка `Удалить` удаляет КП и связанные файлы.
- За 5 дней до автоматического удаления в списке появляется предупреждение.
